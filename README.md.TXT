Olá, vamos aprender a publicar seu primeiro projeto do Git.

1-Link do Git: https://git-scm.com/downloads
2- Vídeo inicial da explicação:https://www.youtube.com/watch?v=UBAX-13g8OM&list=PLhkO7OMKgT_rqwGYldqcFxyN4yjFgmDh8
 

Intruções

Crie uma pasta com o nome "Projeto Git"
Crie um arquivo README.md(markdown), abra no Visual studio Code de preferência.
Faça as anotações e apresentações que julgar necessárias

Utilizando o Git Bash ou o terminal do Visual Code, abra a pasta "Projeto Git"
Digite os seguintes comandos:
- "git init" (Você vai iniciar um repositório vazio .git , que está oculto)
 -"git add README.md" 
-git commit -m "Primeiro commit"(o m é de mensagem)



****No GitHub, vá em Profile>Repository>New
Nomeie o repositório e coloque uma descrição e crie o repositório
Copie o link do seu projeto
Vá para o Git Bash

-git remote add origin cole o link do seu projeto (esse comando faz a conexão do repositório local com o do Github)
-git push -u origin main

faça seu login no Github na aba que vai abrir

*No Github, de refresh e veja seu commit!!!!

Repita o processo do push sempre que adicionar um commit
